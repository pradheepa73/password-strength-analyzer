🔐 AI-Powered Password Strength Analyzer

A professional cybersecurity-based web application that analyzes password strength in real time using Python Flask and the zxcvbn password security library.


---

🚀 Features

✅ Real-time password strength analysis
✅ Strength meter with color indicators
✅ Password crack-time estimation
✅ Suggestions for stronger passwords
✅ Warning detection for weak passwords
✅ Modern cybersecurity-themed UI
✅ Responsive design
✅ Built using Flask + JavaScript


---

🔥 Flask

⚙️ Installation

Clone the repository

git clone https://github.com/pradheepa73/password-strength-analyzer.git

Navigate to project folder

cd password-strength-analyzer

Create virtual environment

python -m venv venv

Activate virtual environment

Windows

venv\Scripts\activate

Mac/Linux

source venv/bin/activate

Install dependencies

pip install -r requirements.txt


---

▶️ Run the Application

python app.py

Open in browser:

http://127.0.0.1:5000
